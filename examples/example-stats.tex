The \texttt{rossstats} package provides consistent APA-style formatting
for common statistical quantities. Every command below can be used in
running text or inside tables.

\subsection*{Example in running text}

The treatment group showed significantly lower relapse rates
(\RR{0.72}{0.55}{0.94}; \pval{0.015}), corresponding to an
\NNT{8}. Diagnostic accuracy was acceptable
(\sensspec{0.92}{0.85}; \ppvnpv{0.88}{0.90}).
